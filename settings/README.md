# To use the software...

## (1) Make `.opencorenmr` directory under your home directory.
- Linux: ~/.opencorenmr, or /home/(username)/.opencorenmr  
- Mac OSX: ~/.opencorenmr, or /Users/(username)/.opencorenmr  
- Windows: C:\Users\(username)\.opencorenmr
